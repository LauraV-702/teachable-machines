# 🎮 AI Detection Game (Teachable Machine + Webcam)

A simple browser-based AI game that uses a **Teachable Machine image model** to detect real-time webcam input and turn it into a scoring game.

The game reacts to what the camera sees (phone, no phone, or Chiikawa) and updates your score instantly.

---

## 🚀 Features

- 📷 Real-time webcam detection using TensorFlow.js + Teachable Machine
- 🧠 AI model integration (Google Teachable Machine)
- 🎮 Simple scoring game mechanics
- 🟢 Dynamic background changes based on prediction
---

## 🎯 Game Rules

The AI detects 3 possible states:

- 🟢 **NO PHONE** → +1 point  
- 🔴 **HOLDING PHONE** → -1 point  
- 💖 **CHIIKAWA PRESENT** → +3 points  

Try to maximize your score by adjusting what the camera sees!
